    • Summary: Latitude vs X


    The purpose of this analysis is to visualize how climate changes at varying distances from the equator. We use the OpenWeatherMap API to assemble a dataset of ~500 randomly selected cities at varying latitudes.

    Once the data was assembled, we used MatPlotLib to visualize several factors in relation to latitude: temperature, cloudiness, wind speed, and humidity.

    This site contains the data visualizations from the climate analysis, as well as the raw data used and the step-by-step analysis.
